# منتدى أفكار التطبيقات

منتدى أفكار التطبيقات هو منصة عصرية مصممة لمساعدة رواد الأعمال والمطورين على تحويل الأفكار الغامضة إلى خطط عمل قابلة للتطبيق. باستخدام تقنيات متقدمة، توفر المنصة حلاً شاملاً لتطوير وتقييم أفكار التطبيقات.

---

## 🛠️ تقنيات المشروع

### **الواجهة الأمامية**
- **الإطار:** [Next.js 15+](https://nextjs.org/) (App Router)
- **المكتبة:** [React 19](https://react.dev/)
- **التصميم:** [Tailwind CSS 4](https://tailwindcss.com/)
- **الأيقونات:** [Lucide React](https://lucide.dev/)

### **الخادم الخلفي**
- **الإطار:** [FastAPI](https://fastapi.tiangolo.com/) (Python 3.10+)
- **تحليل البيانات:** Pydantic

---

## ⚙️ إعدادات التثبيت المحلي

اتبع هذه الخطوات لتشغيل المشروع على جهازك المحلي.

### **1. التنقل إلى المشروع**
افتح الطرفية (Command Prompt أو PowerShell أو Terminal) وادخل مجلد المشروع:
```bash
cd "path/to/project"
```

### **2. إعداد الخادم الخلفي**
الخادم الخلفي يتعامل مع منطق التطبيق الرئيسي.
1. ادخل مجلد الخادم الخلفي:
   ```bash
   cd backend
   ```
2. ثبت المتعلقات:
   ```bash
   pip install -r requirements.txt
   ```
3. ابدأ السيرفر:
   ```bash
   python -m uvicorn main:app --reload
   ```
   *سيكون API متاحاً على `http://localhost:8000`*

### **3. إعداد الواجهة الأمامية**
الواجهة الأمامية توفر واجهة المستخدم التفاعلية.
1. افتح **نافذة طرفية جديدة** وانتقل إلى جذر المشروع:
   ```bash
   cd "path/to/project"
   ```
2. ثبت المتعلقات:
   ```bash
   npm install
   ```
3. ابدأ خادم التطوير:
   ```bash
   npm run dev
   ```
   *سيكون التطبيق متاحاً على `http://localhost:3000`*

## 📁 هيكل المجلدات

| المجلد/الملف | الغرض |
| :--- | :--- |
| `app/` | يحتوي على صفحات Next.js App Router وتنسيق الأنماط العام. |
| `components/` | عناصر واجهة المستخدم القابلة لإعادة الاستخدام. |
| `backend/` | تطبيق FastAPI من Python يحتوي على منطق الخادم. |
| `backend/main.py` | نقطة الدخول لخادم FastAPI وتعريف المسارات. |
| `lib/` | وظائف الأدوات المشتركة وثوابت الإعدادات. |
| `public/` | الأصول الثابتة مثل الصور والخطوط. |
| `.env.local` | متغيرات بيئة الواجهة الأمامية. |

## ✨ المميزات الرئيسية

1. **واجهة احترافية**: لوحة عمل حديثة مع تصاميم سريعة الاستجابة وانتقالات سلسة.
2. **إدارة الأفكار**: نظام متقدم لتنظيم وتصفية أفكار التطبيقات.
3. **تقييم شامل**: تحليل تفصيلي لكل فكرة تطبيق مع مسارات متعددة للتطوير.
4. **نظام التفاعل**: واجهة تفاعلية لتقديم ردود فعل وملاحظات على الأفكار.
